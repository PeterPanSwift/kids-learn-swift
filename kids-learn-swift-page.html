🍎什麼是 Swift？

Swift 是一種超級酷的程式語言！它就像是一種魔法，可以讓你創造出奇妙的 iPhone 和 iPad 應用程式。

有趣的事實： Swift 的標誌是一隻飛快的小鳥，因為 Swift 這個字的意思就是「快速」！

🎨用 Swift 畫畫

想像一下，你可以用 Swift 來畫畫！你可以創造各種形狀、顏色，甚至是動畫！

// 這是一個畫圓形的例子 let circle = UIView(frame: CGRect(x: 100, y: 100, width: 100, height: 100)) circle.backgroundColor = .red circle.layer.cornerRadius = 50 view.addSubview(circle)

🎮製作自己的遊戲

使用 Swift，你可以創造出自己的遊戲！想像一下，你可以設計出一個太空冒險或是一個可愛的寵物照顧遊戲。

🌈為什麼學習 Swift 很棒？

  • 你可以創造出自己的 App
  • 它很有趣，就像解謎遊戲
  • 你可以和朋友分享你的作品
  • 它能幫助你更好地理解科技
小提示： 記住，每個偉大的程式設計師都是從小小的步伐開始的。慢慢來，保持好奇心，你一定會成功的！
